================================================================================
  КАРТА ЛЕГЕНДАРНЫХ ЖИВОТНЫХ RDO — ИНСТРУКЦИЯ
================================================================================

Интерактивная карта и таблица для охоты на легендарных животных в
Red Dead Online. Показывает, где искать зверя, в какое игровое время и при
какой погоде он доступен, помогает вести прогресс и отслеживать кулдаун
по виду.

Игровое время считается так же, как на RDO Map (в 30 раз быстрее реального):
https://jeanropke.github.io/RDOMap/


--------------------------------------------------------------------------------
КАК УСТРОЕН ЭКРАН
--------------------------------------------------------------------------------

Страница разделена на три части:

  Слева      — фильтры, часы, легенда цветов
  По центру  — таблица всех животных
  Справа     — карта мира

Внизу карты — строка состояния с игровым временем, погодой и версией карты.


--------------------------------------------------------------------------------
ВРЕМЯ И ПОГОДА
--------------------------------------------------------------------------------

ИГРОВЫЕ ЧАСЫ

  В левой панели отображается текущее игровое время RDO. Оно обновляется
  автоматически. Ориентируйтесь на часы на этой странице, а не на часы
  в углу экрана игры.

ПОГОДА

  Выберите в списке «Погода» ту погоду, которая сейчас в вашей сессии:

    - Ясно
    - Туман
    - Буря
    - Дождь

  От выбранной погоды зависит, какие животные считаются доступными прямо сейчас.


--------------------------------------------------------------------------------
КАРТА
--------------------------------------------------------------------------------

ЦВЕТА ЗОН

  На карте у каждого животного есть цветная зона. Расшифровка — слева в блоке
  «Цвета»:

    Зелёный           — можно искать сейчас (подходят время и погода)
    Жёлтый/оранжевый  — не подходит время или погода
    Красный           — животное на кулдауне
    Голубой           — все пункты прогресса отмечены (выполнено)

ЭЛЕМЕНТЫ НА КАРТЕ

    - Большой круг — зона появления животного
    - Иконка в центре зоны — для узнавания вида
    - Красные точки — места появления (спавны).
      КЛИК ПО КРАСНОЙ ТОЧКЕ открывает подсказку с подробностями
    - Значок быстрого перемещения — точка fast travel
      (можно скрыть фильтром слева)

ПОДСКАЗКА НА КАРТЕ

  При клике на красную точку открывается окно:

    - название и категория животного
    - время появления (например: 09:00 - 18:00)
    - нужная погода
    - статус (можно искать / не подходит / кулдаун / выполнено)
    - для активных животных: «Исчезнет через: …» — сколько осталось
      до конца игрового окна появления
    - прогресс — кнопки для отметок
    - кулдаун — поле начала, кнопки «72ч» и «ВЫКЛ»

  Выбранное животное подсвечивает строку в таблице ЖЁЛТЫМ.
  Активные животные в таблице дополнительно подсвечены ТУСКЛО-ЗЕЛЁНЫМ.


--------------------------------------------------------------------------------
ТАБЛИЦА
--------------------------------------------------------------------------------

  Таблица содержит ВСЕ легендарные животные, в том числе те, у которых нет
  зоны на карте, которые ищутся по заданиям Гарриет.

СТОЛБЦЫ

  Животное
    Название. КЛИК ПО ИМЕНИ — карта перейдёт к животному и откроет подсказку
    (если оно есть на карте).

  Время
    Часы появления в игровом времени.

  Погода
    Нужная погода.

  Отслежено … Одежда
    Восемь ячеек прогресса. КЛИК ПО ЯЧЕЙКЕ ставит или снимает отметку «+».

  Начало кулдауна
    Дата и время начала (можно изменить вручную).

  Окончание
    Автоматически: начало + 72 часа.

  Кулдаун
    Кнопки «72ч» (включить) и «ВЫКЛ» (выключить).

ПОДСВЕТКА СТРОК

    Тускло-зелёная  — животное сейчас активно
    Жёлтая          — выбранное животное (открыта подсказка или клик по имени)
    Красная полоса  — включён кулдаун
    Зелёное имя     — все 8 пунктов прогресса отмечены

ПРОГРЕСС

  Отмечайте действия в таблице (клик по «+») или в подсказке на карте.
  Отметки синхронизируются между таблицей и подсказкой.

  Животное считается ВЫПОЛНЕННЫМ, когда отмечены все 8 пунктов:
  отслежено, убито, освежевано, изучено, усыплено, образцы, фото, одежда.


--------------------------------------------------------------------------------
КУЛДАУН
--------------------------------------------------------------------------------

  Кулдаун длится 72 ЧАСА (реального времени) и действует на ВЕСЬ ВИД,
  а не на одну разновидность.

  Пример: кулдаун на «Бизона Татанка» действует и на «Бизона Виньяна»,
  и на «Бизона Пэйта» — это один вид «Бизон».

КАК ВКЛЮЧИТЬ

  1. Укажите начало кулдауна (или оставьте текущее время).
  2. Нажмите «72ч» в таблице или в подсказке.

  Столбец «Окончание» пересчитается автоматически. На карте зона станет красной.

КАК ВЫКЛЮЧИТЬ

  Нажмите «ВЫКЛ» — кулдаун снимется для всего вида.


--------------------------------------------------------------------------------
ФИЛЬТРЫ (ЛЕВАЯ ПАНЕЛЬ)
--------------------------------------------------------------------------------

УСЛОВИЯ

  Только доступные
    На карте остаются только зелёные (активные сейчас) животные.

  Скрыть на кулдауне
    Убрать с карты животных на кулдауне.

  Скрыть выполненные на карте
    Убрать с карты полностью выполненных.

  Быстрое перемещение
    Показать или скрыть точки fast travel.

КАТЕГОРИИ

  Тёмные, Белые, Рыжие и светлые, Пестрые — можно включать отдельно.

ФИЛЬТР ПРОГРЕССА

  Помогает найти животных по состоянию: незавершённые, без образцов,
  без фото, с конкретной отметкой и т.д. Можно отметить несколько сразу.

СБРОСИТЬ ПРОГРЕСС

  Удаляет ВСЕ отметки прогресса и ВСЕ кулдауны в этом браузере.
  Перед нажатием появится подтверждение. Действие необратимо.


--------------------------------------------------------------------------------
СОХРАНЕНИЕ ДАННЫХ
--------------------------------------------------------------------------------

  Прогресс и кулдауны сохраняются автоматически в браузере.
  После обновления страницы или закрытия вкладки данные остаются.

  ВАЖНО:

    - данные привязаны к этому браузеру на этом устройстве
    - на другом компьютере или в другом браузере прогресс будет пустым
    - очистка данных сайта в браузере удалит все отметки
    - фильтры после перезагрузки сбрасываются к умолчанию


--------------------------------------------------------------------------------
СОВЕТЫ
--------------------------------------------------------------------------------

  1. Сверяйте погоду в игре с выбранной в фильтре.
  2. Для быстрого поиска включите «Только доступные».
  3. После взаимодействия со зверем сразу ставьте кулдаун «72ч».
  4. Кликайте по имени в таблице, чтобы перейти к животному на карте.

================================================================================
  RDO LEGENDARY ANIMALS MAP — USER GUIDE
================================================================================

An interactive map and table for hunting legendary animals in Red Dead Online.
Shows where to find each animal, at what in-game time and weather it is
available, helps track progress and species cooldowns.

In-game time works like RDO Map (30x faster than real time):
https://jeanropke.github.io/RDOMap/


--------------------------------------------------------------------------------
SCREEN LAYOUT
--------------------------------------------------------------------------------

The page has three sections:

  Left    — filters, clock, color legend
  Center  — table of all animals
  Right   — world map

At the bottom of the map — status bar with in-game time, weather, and map version.


--------------------------------------------------------------------------------
TIME AND WEATHER
--------------------------------------------------------------------------------

IN-GAME CLOCK

  The left panel shows the current RDO in-game time. It updates automatically.
  Use the clock on this page, not the clock in the corner of your game screen.

WEATHER

  In the Weather dropdown, select the weather in your current session:

    - Clear
    - Foggy
    - Storm
    - Rain

  Which animals are available right now depends on the selected weather.


--------------------------------------------------------------------------------
MAP
--------------------------------------------------------------------------------

ZONE COLORS

  Each animal has a colored zone on the map. Legend is in the Colors block on
  the left:

    Green           — available now (time and weather match)
    Yellow/orange   — wrong time or weather
    Red             — animal is on cooldown
    Blue            — all progress items checked (completed)

MAP ELEMENTS

    - Large circle — animal spawn zone
    - Icon in the center — species identifier
    - Red dots — spawn points. CLICK A RED DOT to open the info popup
    - Fast travel icon — fast travel point (can be hidden with the left filter)

MAP POPUP

  Clicking a red spawn dot opens a window with:

    - animal name and category
    - spawn time (e.g. 09:00 - 18:00)
    - required weather
    - status (available / not suitable / cooldown / completed)
    - for active animals: "Disappears in: …" — time left until the spawn window ends
    - progress — buttons to mark tasks
    - cooldown — start field, "72h" and "OFF" buttons

  The selected animal highlights its table row in YELLOW.
  Active animals are also highlighted in DIM GREEN in the table.


--------------------------------------------------------------------------------
TABLE
--------------------------------------------------------------------------------

  The table lists ALL legendary animals, including those without a map zone
  that are found through Harriet's missions.

COLUMNS

  Animal
    Name. CLICK THE NAME — the map pans to the animal and opens the popup
    (if it is on the map).

  Time
    Spawn hours in in-game time.

  Weather
    Required weather.

  Tracked … Garment
    Eight progress cells. CLICK A CELL to toggle a "+" mark.

  Cooldown start
    Start date and time (can be edited manually).

  Cooldown end
    Automatic: start + 72 hours.

  Cooldown
    "72h" (enable) and "OFF" (disable) buttons.

ROW HIGHLIGHTING

    Dim green   — animal is active right now
    Yellow      — selected animal (popup open or name clicked)
    Red bar     — cooldown enabled
    Green name  — all 8 progress items checked

PROGRESS

  Mark tasks in the table (click "+") or in the map popup. Marks sync between
  the table and popup.

  An animal is COMPLETED when all 8 items are checked: tracked, killed,
  skinned, studied, sedated, samples, photo, garment.


--------------------------------------------------------------------------------
COOLDOWN
--------------------------------------------------------------------------------

  Cooldown lasts 72 HOURS (real time) and applies to the entire SPECIES,
  not a single variant.

  Example: cooldown on Tatanka Bison also applies to Winyan Bison and
  Payta Bison — they are all one species "Bison".

HOW TO ENABLE

  1. Set cooldown start (or leave the current time).
  2. Click "72h" in the table or popup.

  The Cooldown end column updates automatically. The map zone turns red.

HOW TO DISABLE

  Click "OFF" — cooldown is cleared for the whole species.


--------------------------------------------------------------------------------
FILTERS (LEFT PANEL)
--------------------------------------------------------------------------------

CONDITIONS

  Only available
    Show only green (active now) animals on the map.

  Hide on cooldown
    Remove animals on cooldown from the map.

  Hide completed on map
    Remove fully completed animals from the map.

  Fast travel
    Show or hide fast travel points.

CATEGORIES

  Dark, Light, Red and blond, Patterned — toggle individually.

PROGRESS FILTER

  Find animals by progress state: incomplete, missing samples, missing photo,
  specific marks, etc. Multiple filters can be selected.

RESET PROGRESS

  Deletes ALL progress marks and ALL cooldowns in this browser.
  A confirmation appears first. This cannot be undone.


--------------------------------------------------------------------------------
DATA STORAGE
--------------------------------------------------------------------------------

  Progress and cooldowns are saved automatically in the browser.
  Data remains after refresh or closing the tab.

  IMPORTANT:

    - data is tied to this browser on this device
    - another computer or browser starts with empty progress
    - clearing site data in the browser removes all marks
    - filters reset to default after reload


--------------------------------------------------------------------------------
TIPS
--------------------------------------------------------------------------------

  1. Match in-game weather with the filter selection.
  2. Enable "Only available" for quick hunting.
  3. Set a "72h" cooldown right after interacting with the animal.
  4. Click a name in the table to jump to the animal on the map.


--------------------------------------------------------------------------------
RUNNING ON YOUR OWN PC
--------------------------------------------------------------------------------

  1. Run "Launch map.bat"
     or manually:
       python export_legendaries.py
       python serve_map.py

  2. Open in browser: http://localhost:8765

  This section is for the project owner only. Shared users do not need these files.


--------------------------------------------------------------------------------
PUBLISHING ONLINE
--------------------------------------------------------------------------------

  To share the map, upload the legendary-map folder to a host
  (GitHub Pages, Netlify, etc.).

  Users only need the site link in a browser. They do not need
  "Launch map.bat" or Python. Each user keeps their own progress in their browser.
