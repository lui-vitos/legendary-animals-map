Map and table guide

This map is built specifically for Red Dead Online legendary animal hunting. Unlike general-purpose maps, it focuses on time, weather, spawn areas, and progress tracking.

1. Layout

  • Left — filters and settings.
  • Center — full animals table.
  • Right — interactive map with spawn zones.

2. Time and weather

The left panel shows RDO game time (synced with RDO Map).

Select current in-game weather: Clear, Foggy, Storm, or Rain.

3. Main filters

  • Only available — shows animals matching current time and weather.
  • Hide on cooldown — hides animals on species cooldown.
  • Hide completed on map — hides fully completed animals.
  • Fast travel — toggles fast travel points.

4. Categories

  • Dark
  • Light
  • Red and blond
  • Patterned

Category filters affect the map. The table always shows all animals, but status highlighting updates.

5. Progress filter

Use this to find animals by your progress state (incomplete, missing samples/photo, etc.).

6. Colors (legend)

ColorMeaning
GreenAvailable now (time + weather match)
Yellow/orangeNot suitable now
RedOn cooldown
BlueCompleted

7. Map interaction

  • Zone — possible spawn area.
  • Center icon — animal marker.
  • Red point — spawn point inside zone.

Click zone/icon to open popup and highlight the table row.

8. Table

  • Click animal name to focus on map.
  • Use “+” buttons to track progress.
  • Cooldown start/end fields support 72h species cooldown.

Some Harriet-only animals appear in table only (no map zone).

9. Progress and cooldown

Progress is saved automatically in browser localStorage.

Cooldown is shared per species (72h). Use 72h to enable and OFF to clear.

10. Quick hunting workflow

  1. Open map and choose language.
  2. Set current weather.
  3. Enable “Only available”.
  4. Pick target on map/table.
  5. Mark progress and cooldown after hunt.
