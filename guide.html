Map and table guide

This map is built specifically for Red Dead Online legendary animal hunting. Unlike general-purpose maps, it focuses on time, weather, spawn areas, and progress tracking.

1. Layout

  • Left — filters and settings.
  • Center — full animals table.
  • Right — interactive map with spawn zones.

1.1. Collapse table panel

On the border between the table and map there is a vertical button labeled “Table” with an arrow.

  • ◀ — hide the table and expand the map (filters stay on the left).
  • ▶ — show the table again.

The button stays on the table/map seam. State is saved in your browser.

2. Time and weather

The left panel shows Red Dead Online game time, synced with the in-game clock.

Weather is a multi-select: Any weather, Clear, Foggy, Storm, Rain. Each label shows the same icon used on the map zones. “Any weather” checks all four types; unchecking one of the four also unchecks “Any”. An animal matches weather if its requirement matches at least one selected option.

By default all weather types are selected. Uncheck ones you do not want.

3. Main filters

  • Only available — shows animals matching current time and selected weather.
  • Hide on cooldown — hides animals on species cooldown.
  • Hide completed on map — hides fully completed animals from the map (all 8 progress marks).
  • Hide completed in table — hides fully completed animals from the table so you only see unfinished ones.
  • Fast travel — toggles fast travel points.

4. Categories

  • Dark
  • Light
  • Red and blond
  • Patterned

Category filters affect the map. The table follows category highlighting and can hide fully completed animals when that filter is enabled.

5. Progress filter

Use this to find animals by your progress state:

  • Incomplete
  • Missing samples
  • Missing photo
  • Individual toggles: Tracked, Killed, Skinned, Studied, Sedated, Samples, Photo, Garment

6. Colors (legend)

ColorMeaning

Yellow/orangeNot suitable now
RedOn cooldown

7. Map interaction

  • Zone — possible spawn area.
  • Center icon — animal marker.
  • Red point — spawn point inside zone.

Click zone/icon to open popup and highlight the table row.

While a spawn window is active by time, the center of a large zone shows a semi-transparent dotted LED-style timer — remaining real-world minutes until the window ends — plus a weather icon for that animal. If it is not spawn time, the timer is hidden.

8. Table

  • Click animal name to focus on map.
  • Use “+” buttons to track progress.
  • Cooldown start/end fields support 72h species cooldown.
  • Progress columns: Tracked, Killed, Skinned, Studied, Sedated, Samples, Photo, Garment.

Some Harriet-only animals appear in table only (no map zone).

9. Progress and cooldown

Mark progress: Tracked, Killed, Skinned, Studied, Sedated, Samples, Photo, Garment. When all 8 are checked, the animal is complete.

Progress is saved automatically in your browser.

Cooldown is shared per species (72h). Use 72h to enable and OFF to clear.

10. Quick hunting workflow

  1. Open map and choose language.
  2. Select matching weather (multi-select allowed).
  3. Enable “Only available”.
  4. Optionally enable “Hide completed in table”.
  5. Pick target on map/table.
  6. Mark progress and cooldown after hunt.

11. Free Roam Events

At the bottom of the map there are two banners:

  • Free Roam Events with legendary animals — nearest legendary event, today’s plan (4 slots), link to legendary schedule.
  • Free Roam Events (general) — nearest general and role events, link to full schedule.

Event times use your real local time (not RDO game time). Time zone is detected automatically.

Hover over an event row in a banner for a description tooltip. The nearest legendary event is highlighted and marked (Nearest).

Recommended ability card loadouts for events and Showdown modes: Recommended builds (see §11.5).

11.2. Legendary animal event sounds

  • Warn … min before — checkbox and number of minutes (configurable, default 10). Two short tones.
  • Sound at start — separate longer signal when the event begins (on by default).
  • Enable alerts for Protect Legendary Animal, Wild Animal Tagging, or both.
  • Test warning / Test start — preview sounds without waiting.

Settings saved in browser. Click the map once to unlock audio. The tab must stay open.

11.3. Map banners

  • All Free Roam Events — show or hide the general banner.
  • Events with legendary animals — show or hide the legendary banner and day plan.

11.5. Recommended ability builds

A separate page lists ability card loadouts and tips for every Free Roam Event and Showdown Series mode (not the same as FME): Recommended builds.

How to use

  • At the top — templates (PvP, PvE, bow, tank, etc.). Click one to highlight cards on the right. Click again to clear.
  • Each FME event and Showdown mode lists several matching templates — pick what fits your playstyle.
  • Gold border — required card; cyan dashed border with OR — pick one of several options.
  • Hover a card on the right for its Tier III description (in-game text).
  • The gallery follows RDO categories: Dead Eye, Combat, Recovery, Defense.

You equip 4 cards in RDO: 1 Dead Eye + 3 passive (slots at ranks 10, 20, 40). Upgrade cards to Tier III for full effect. Tips are community-sourced, not official Rockstar guides.

Free Roam Events on the builds page

All 17 scheduled events (see full schedule), including Random Challenge variants (bow kills, headshots, horseback, longarm, sidearm, etc.).

Showdown Series
